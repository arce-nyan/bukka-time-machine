# 物価タイムマシン

その値段、いつの日本？  
物価帳に並ぶ値段から、その年を推理する年代当てクイズです。

[▶ ゲームを遊ぶ](https://arce-nyan.github.io/bukka-time-machine/)

## 遊び方

1955年から2024年までの10年分の物価データから、毎回1年分をランダムに出題します。値段を手がかりに西暦を選び、「この年で確定」を押してください。

誤差2年以内なら100点満点です。以降は1年離れるごとに6点ずつ減点されます。ヒントを使うと、その問題の得点は半分になります。

## 出題項目

- 国家公務員・大卒初任給
- 米 5kg
- 卵（卸売・Mサイズ）
- 新聞 月極購読料
- ラーメン（外食）
- 映画 入場料
- 銭湯（東京・大人）
- ガソリン 1L

## データについて

各品目は公的統計、業界団体資料、一次資料・関連文献をもとに収集しています。詳細な出典と採用方法は、ゲーム内の「ⓘ データの出典を見る」から確認できます。

品目ごとに地域、単位、調査時点は異なります。たとえば卵は卸売相場、銭湯は東京都の統制料金です。そのため、このゲームの値段は厳密な家計比較ではなく、当時の年代を感じ取るための手がかりとして扱っています。

ラーメンなど一部の値は、近接年の実測値や価格指数から補った参考値を含みます。該当する値は答え合わせ後に `※` で示します。

## 技術・プライバシー

静的な単一HTMLページとして動作し、ビルドやサーバーは不要です。外部APIは利用していません。Google Fontsを読み込み、最高スコアのみをブラウザ内に保存します。

## ローカルで試す

リポジトリを取得後、`index.html` をブラウザで開いてください。
